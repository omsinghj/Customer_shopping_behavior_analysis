🛍️ Customer Shopping Behavior Analysis

📌 Project Overview

This project analyzes customer shopping behavior using transactional data to uncover actionable insights that improve:

* 📈 Sales performance
* 🎯 Customer engagement
* 🔁 Retention & loyalty

The analysis is based on 3,900 customer transactions across multiple product categories, combining Python, SQL, and Power BI to deliver end-to-end business in0sights.

🎯 Business Problem

A retail company wants to better understand:

* Changing customer purchase patterns
* Impact of discounts, reviews, and subscriptions
* Differences across demographics and channels

👉 Goal:
Leverage data to optimize marketing strategies, improve customer experience, and increase revenue.

🧰 Tech Stack

| Tool                      | Purpose                       |
| ------------------------- | ----------------------------- |
| 🐍 Python (Pandas, NumPy) | Data cleaning & preprocessing |
| 🐘 PostgreSQL             | Data storage & SQL analysis   |
| 📊 Power BI               | Dashboard & visualization     |
| 📓 Jupyter Notebook       | Development & analysis        |



🔄 Project Workflow

1️⃣ Business Understanding
* Defined key business objectives
* Identified important metrics:
    * Revenue
    * Customer segments
    * Purchase behavior drivers

2️⃣ Data Preparation (Python)
* Loaded dataset using pandas
* Performed:
     * Data inspection (info(), describe())
     * Missing value handling
     * Column standardization

✔️ Key Transformations:

* Imputed missing review_rating values using median
* Created age_group feature
* Created purchase_frequency_days
* Removed redundant columns

3️⃣ Database Integration (PostgreSQL)
* Connected Python to PostgreSQL
* Created structured customer table
* Enabled scalable querying

4️⃣ Data Analysis (SQL)

Performed business-driven analysis:

* 💰 Revenue by gender
* 🛒 High-value discount users
* ⭐ Top-rated products
* 🚚 Shipping impact on spending
* 📦 Subscription vs non-subscription behavior
* 🎯 Customer segmentation:
      * New
      * Returning
      * Loyal
* 📊 Revenue by age group

5️⃣ Data Visualization (Power BI)

Built an interactive dashboard including:

* Revenue insights
* Customer segmentation
* Product performance
* Subscription impact

📌 Designed for business stakeholders & decision-making

📊 Key Insights

* 👩 Female customers generate slightly higher revenue
* 🚚 Express shipping users spend more per transaction
* 💳 Subscribers contribute significantly higher revenue
* 🏷️ Discounts attract high-value “smart shoppers”
* 🔁 Loyal customers drive long-term business growth

💡 Business Recommendations

* 🎯 Promote subscription programs
* 🎁 Implement customer loyalty rewards
* ⚖️ Optimize discount strategies
* 🌟 Highlight top-rated products
* 📢 Use targeted marketing campaigns
